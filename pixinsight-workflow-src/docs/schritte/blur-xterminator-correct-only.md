# BlurXTerminator Correct Only

<div class="step-icon-clear"></div>

## Ziel

- Optimierung des Bildes für weitere Prozesschritte wie Plate Solving und SPCC.

### Was „Correct Only“ macht:

- Korrigiert Koma
- Korrigiert Astigmatismus
- Korrigiert leichte Defokus-Fehler
- Korrigiert Chromatische Aberration
- Korrigiert Guiding-/Bewegungsunschärfe in begrenztem Umfang
- Macht verzogene Sterne runder
- Vereinheitlicht die Sternabbildung über das gesamte Bildfeld

### Was „Correct Only“ nicht macht:

- Keine Deconvolution-Schärfung von Sternen
- Keine Schärfung von Galaxien- oder Nebeldetails
- Keine Wiederherstellung feiner Strukturen
- Keine Erhöhung des lokalen Kontrasts durch Schärfung
